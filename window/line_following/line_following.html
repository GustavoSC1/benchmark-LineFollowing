Este simulação visa criar um algoritmo robusto e eficiente para que o robô L1R2 consiga evitar obstáculos e completar o percusso, utilizando a linguagem de programação Python.

O objetivo é que o robô atravesse a sala e alcance o ladrilho de chegada o mais rápido possível, superando todos os desafios pelo caminho.

t = 00:00:00

A métrica de referência t é o tempo necessário para o robô atravessar a sala. Minimizar esse tempo é o objetivo para este cenário.

O cronômetro para depois que o robô chega no ladrilho de chegada.

Programando um robô em Python

  1. Abra o editor de código-fonte: clique no robô com o botão direito do mouse para abrir o menu contextual do robô e selecione o item "Editar controlador". Uma janela deve abrir contendo o código fonte do controlador do robô. Este é realmente um programa Python simples.
  2. Edite o controlador: na linha 7, modifique o valor atribuído à variável de destino: em vez de 0, tente definir um valor diferente, por exemplo, 10.
  3. Salve seu programa: no editor de código-fonte, pressione Ctrl-S para salvar seu programa.
  4. Reinicializar a simulação: pressione o botão de reinicialização para levar em consideração suas modificações.
  5. Executar a simulação: pressione o botão play para executar a simulação.
  6. Observe o resultado: o robô deve avançar. Seu desempenho deve ser atualizado em tempo real na guia Métricas. Depois de um tempo, o robô deve parar e seu desempenho deve ser registrado (se você estiver logado). Tente modificar o programa do controlador novamente para obter um melhor desempenho.

Durante a execução, o controlador de amostra imprime alguma saída no console. Você pode abrir o console clicando no botão da barra de ferramentas.
